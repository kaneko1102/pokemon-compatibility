# ポケモンの技の相性を表示
## アプリケーションの概要
ポケモンのタイプと技のタイプから相性を表示する

## 開発環境
- 言語: JavaScript
- 実行環境: Deno

## 実行方法
コマンドラインで
```
deno run -A pokemon.js
```
と入力し，[http://localhost:8000](http://localhost:8000)にアクセス
